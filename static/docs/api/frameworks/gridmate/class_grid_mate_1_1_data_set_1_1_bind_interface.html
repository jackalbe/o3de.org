Detailed Description

template<typename DataType, typename MarshalerType = Marshaler<DataType>, typename ThrottlerType = BasicThrottle<DataType>>
template<class C, void(C::*)(const DataType &, const TimeContext &) FuncPtr, typename CallbackTraits>
class GridMate::DataSet< DataType, MarshalerType, ThrottlerType >::BindInterface< C, FuncPtr, CallbackTraits >

Declares a DataSet with an event handler that is called when the DataSet is changed. Use BindInterface<Class, FuncPtr> to dispatch to a method on the ReplicaChunk's ReplicaChunkInterface event handler instance.
