Detailed Description

Carrier driller

Note
Be careful which buses you attach. The drillers work in Multi threaded environment and expect that a driller mutex (DrillerManager::DrillerManager) will be automatically locked on every write. Otherwise in output stream corruption will happen (even is the stream is thread safe).
