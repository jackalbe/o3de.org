template<typename Type>
class GridMate::Marshaler< Type, typename AZStd::Utils::enable_if_c< IsContainerMarshaler< Type >::Value >::type >

Writes a container to the stream. This includes string, vector, map, set, list, both ordered and unordered versions. Assumes the value and key (if exists) has an appropriate marshaler defined.
