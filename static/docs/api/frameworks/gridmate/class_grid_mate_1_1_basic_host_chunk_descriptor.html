Detailed Description

template<typename ReplicaChunkType>
class GridMate::BasicHostChunkDescriptor< ReplicaChunkType >

BasicHostChunkDescriptor implements a helper descriptor that prevents chunk proxies from being created on the host. The idea is to prevent a malicious client from creating (and owning) chunk types that should always be authoritative on the host.
