Detailed Description

template<class Key, AZStd::size_t NumLocks = 8, class Hasher = AZStd::hash<Key>, class EqualKey = AZStd::equal_to<Key>, class Allocator = AZStd::allocator>
class AZStd::concurrent_unordered_multiset< Key, NumLocks, Hasher, EqualKey, Allocator >

Concurrent unordered multiset container, similar to unordered_multiset, but uses a striped array of locks to allow concurrent access.
