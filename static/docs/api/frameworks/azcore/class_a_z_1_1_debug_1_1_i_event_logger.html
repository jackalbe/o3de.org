Utility function to begin an event with a specific structure. For example this can be used as: struct ThreadInfo { uint64_t m_threadId; uint64_t m_processorId; }; auto& info = RecordEventBegin<ThreadInfo>("ThreadInfo"); info.m_threadId = ...; info.m_processorId = ...; RecordEventEnd();

Parameters
idId that uniquely identifies this event.
flagsOptional flags unique to the event.
Returns
A reference of the provided type to store event information in.

virtual void* AZ::Debug::IEventLogger::RecordEventBegin ( EventNameHash  id,
uint16_t  size,
uint16_t  flags = 0 
)
pure virtual

Starts a new event. If there is not enough room left in the thread local buffer then the buffer will be stored to disk and cleared.

Parameters
idId that uniquely identifies this event.
sizeThe total size of the event, excluding the event header. Typically this is the size of the structure that describes the event.
flagsOptional flags unique to the event. For instance a "Thread" event can use the flags to indicate whether the the thread is starting or stopping.
Returns
A void pointer to reserved data in the thread local buffer to write to.
