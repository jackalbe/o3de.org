Detailed Description

DOM like parser, we will load the entire stream in memory (ProcessStream function). Depending on the data size this can be very memory consuming.

Note
DOM is used as reference we are NOT compliant with the DOM specs in any way. IMPORTANT: All data is stored (for parsing) in the same order the events occurred or the remote machine. Each next tad or data was recorded in the way. You can use this as an event index.
