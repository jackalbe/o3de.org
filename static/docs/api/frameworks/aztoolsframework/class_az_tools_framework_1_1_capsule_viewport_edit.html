Detailed Description

Wraps 2 linear manipulators, providing a viewport experience for modifying the radius and height of a capsule. It is designed to be usable either by a component mode or by other contexts which are not associated with a particular component, so it does not contain any reference to an EntityComponentIdPair or other component-based identifier.
