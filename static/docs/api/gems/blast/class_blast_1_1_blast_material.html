float Blast::BlastMaterial::GetStressAngularFactor ( ) const

Factor with which angular stress is applied to destructible objects. Angular stress is calculated based on angular velocity of an object (only non-static actors).

◆ GetStressLinearFactor()

float Blast::BlastMaterial::GetStressLinearFactor ( ) const

Factor with which linear stress is applied to destructible objects. Linear stress includes direct application of BlastFamilyDamageRequests::StressDamage, collisions and gravity (only for static actors).
