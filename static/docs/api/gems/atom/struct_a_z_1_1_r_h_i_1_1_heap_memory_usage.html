Detailed Description

Tracks memory usage for a specific heap in the system. The data is expected to adhere to the following constraints: 1) Reserved <= Budget (unless the budget is 0). 2) Resident <= Reserved.

bool AZ::RHI::HeapMemoryUsage::TryReserveMemory ( size_t  sizeInBytes)
inline

This helper reserves memory in a thread-safe fashion. If the result exceeds the budget, the reservation is safely reverted and false is returned. otherwise, true is returned. Only m_reservedInBytes is affected.
